2. Services Description

2.1 Member Link Overview

Member Link is a membership management solution that integrates with Clover POS systems. The service consists of two components:

Member Link Web Dashboard (memberlink.club)

  • Web-based interface for managing member records
  • Create, edit, and delete member profiles
  • Track membership status, expiration dates, and renewal history
  • Synchronize member data with your Clover customer database
  • Create and manage membership packages and pricing
  • View member activity and order history

Member Link Android App (Clover POS)

  • Barcode scanning for quick member lookup at checkout
  • Assign members to Clover orders during transactions
  • Display membership status and expiration information
  • Add membership renewal items to orders
  • Real-time synchronization with Member Link backend

3.1 Order Data Collection

Member Link collects and stores ALL order data from your Clover merchant account. This is required for the service to function properly. Order data collection includes:

  • All order transactions

  • All line items, quantities, and prices
