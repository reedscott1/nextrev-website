## Local Development

To test locally:
1. Clone this repository
2. Open `index.html` in a web browser
3. Or use a simple HTTP server:
   ```bash
   python -m http.server 8000
   ```
4. Navigate to http://localhost:8000

## Updating Content

### To Update Pages:
1. Edit the HTML files directly
2. Commit and push changes to GitHub
3. GitHub Pages will automatically rebuild (takes 1-2 minutes)

### To Update Legal Documents:
1. Edit `privacy.html` and/or `terms.html`
2. Replace [BRACKETED] placeholders
3. Remove draft notices when finalized
4. Commit and push
